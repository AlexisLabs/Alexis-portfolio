
To run the development server:

```bash

```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

To create an optimized production build:

```bash

```


## ğŸ¨ Customization

### Updating Content

- **Home Page**: Edit `liquid/src/app/page.js` and `liquid/src/app/componets/header.jsx`
- **Resume**: Update `liquid/src/app/resume/page.jsx`
